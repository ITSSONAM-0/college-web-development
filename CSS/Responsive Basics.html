Responsive Basics – Media Queries

Responsive design means webpages adjust automatically to look good on all devices: desktop, tablet, and mobile.

Media Queries are CSS rules that apply styles only if certain conditions (like screen size) are true.

        @media (condition) {
    /* CSS rules here */}

Common Breakpoints (industry standard)

  • max-width: 600px → Mobile
  • max-width: 768px → Tablet
  • max-width: 1024px → Small laptops
  • min-width: 1200px → Large screens
